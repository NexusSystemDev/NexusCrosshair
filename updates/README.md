# Nexus Crosshair Pro Updates

Prepared update targets:

- `github-releases/` for GitHub Releases artifacts generated by electron-builder.
- `generic-server/` for a self-hosted update endpoint.

Publishing is configured in `package.json` through electron-builder `publish`.

Runtime update checks are intentionally disabled by default while the GitHub
repository/update server is not public. Enable them only after publishing a real
release feed:

```powershell
$env:NEXUS_UPDATES_ENABLED="1"
```

For production installers, set `NEXUS_UPDATES_ENABLED=1` in the launch
environment or remove the guard in `src/main/updater.ts` after GitHub Releases
or the generic update server is ready.
